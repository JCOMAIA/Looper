# Looper - Browser Extension

A Chrome-based browser extension for Brave (and other Chromium browsers).

## How to Install

1. Open the **Extensions** tab in Brave (or any Chromium-based browser)
2. Enable **Developer Mode** (toggle in the top right)
3. Click **Load Unpacked**
4. Select the **LOOPER** folder
5. Activate **LOOPER** from the extensions list

## Usage

Once installed and activated, simply use it on any website.
